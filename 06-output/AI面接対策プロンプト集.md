# AI面接対策プロンプト集

面接準備を効率化し、質の高いアウトプットを得るための3つのプロンプトです。

---

## プロンプト1：自分情報の読み込み

AIに自分の強みや経験を深く理解させ、あなたに合った質の高い回答を生成できるようにする

```
【実績・成果物】
[具体的な実績や数値、成果物のリストをペースト]

- プロジェクトA：〇〇を達成、〇〇%改善
- プロジェクトB：〇〇機能を設計・実装、〇〇人のチームをリード
- 技術スタック：〇〇、〇〇、〇〇

【自己分析（強み・弱み）】
[自分の強みと弱みを箇条書きでペースト]
強み：
- 
- 
- 

弱み：
- 
- 

【これまでのキャリアのストーリー】
[なぜこの仕事を選んだのか、どんなきっかけがあったのか、どう成長してきたかを記述]

【大切にしている価値観】
[仕事で大切にしている価値観、働く上で譲れないこと]

# 分析してほしいこと
上記の情報をもとに、以下を分析してください：

1. **私の最大の強みトップ3**：面接で最もアピールすべきポイント
2. **印象に残るユニークな特徴**：他の候補者と差別化できるポイント
3. **物語性のある経験**：面接官の記憶に残るエピソード
4. **弱みをどう伝えるべきか**：誠実さを保ちながら前向きに伝える方法
5. **私のキャリアの一貫性**：これまでの経験に共通するテーマやストーリー
```

## プロンプト3：3×3志望動機と想定問答の生成

### 目的
企業を3つ褒め、自分の強み3つをアピールする完璧な志望動機を作成し、想定問答を準備する

### プロンプト

```
# あなたの役割
あなたは優秀なキャリアアドバイザーです。私の情報と企業情報をもとに、面接で刺さる志望動機と想定問答を作成してください。

# 前提条件
すでに以下の情報を理解していることを前提とします：
- 私の強み、経験、実績（プロンプト1で読み込み済み）
- 志望企業の求める人材像、文化、特徴（プロンプト2で読み込み済み）

もし上記の情報が不足している場合は、改めて私に質問してください。

---

# タスク1：3×3志望動機の作成

以下のルールに従って、志望動機を作成してください。

## 作成ルール
1. **企業を3つ褒める**：企業の良いところを具体的に3つ挙げる
   - 事業内容、企業文化、ビジョン、チームの雰囲気、働き方など
   - 抽象的ではなく、具体的に褒める
   
2. **自分の強みを3つアピールする**：その企業で活かせる自分の強みを3つ挙げる
   - 企業が求める人材像にマッチする強みを選ぶ
   - 具体的なエピソードや数値を交えて説明する
   
3. **相性の良さを伝える**：企業と自分の価値観が合致していることを示す

4. **口調**：カジュアルで親しみやすく、かつ誠実な口調で書く

5. **長さ**：1分程度で話せる長さ（300〜400字程度）

## 出力形式
```
【3×3志望動機】

（企業を褒める①）御社の〇〇という点に強く惹かれました。〇〇〇〇〇。

（企業を褒める②）また、〇〇という企業文化も素晴らしいと感じています。〇〇〇〇〇。

（企業を褒める③）さらに、〇〇という取り組みにも共感しました。〇〇〇〇〇。

（自分の強み①）私の強みの一つは〇〇です。〇〇〇〇〇という経験があります。

（自分の強み②）また、〇〇という能力も持っています。〇〇〇〇〇という実績があります。

（自分の強み③）さらに、〇〇という姿勢も大切にしています。〇〇〇〇〇。

（相性の良さ）これらの強みを御社で活かし、〇〇に貢献しながら、自分自身も成長していきたいと考えています。
```

# タスク2：想定問答リストの作成

面接でよく聞かれる質問への回答を、私の言葉に最適化して作成してください。

## 想定質問リスト
以下の質問に対する回答を作成してください：

1. **自己紹介をお願いします**（1分程度）
2. **志望動機を教えてください**（タスク1で作成したものを使用）
3. **転職理由は何ですか？**（ネガティブな理由は前向きに言い換える）
4. **あなたの強みは何ですか？**（具体的なエピソード付き）
5. **あなたの弱みは何ですか？**（誠実に開示し、どう克服しているかを伝える）
6. **これまでで最も困難だった経験と、それをどう乗り越えましたか？**
7. **なぜうちの会社を選んだのですか?**（他社との違いを明確に）
8. **第一志望ですか？**（即答できるように準備）
9. **他社の選考状況は？**（正直に、かつ前向きに）
10. **5年後のキャリアプランは？**
11. **逆質問：何か質問はありますか？**（3つ程度準備）

## 出力形式
各質問に対して、以下の形式で回答を作成してください：

```
### 質問：〇〇〇〇〇

【キーワード3つ】
1. 〇〇
2. 〇〇
3. 〇〇

【回答例（自分の言葉で話す際の参考）】
〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇。

【ポイント】
- 〇〇を強調する
- 〇〇という点を具体的に伝える

## 使い方のフロー

### ステップ1：自分情報を読み込ませる
プロンプト1を使って、AIに自分の情報を理解させます。

### ステップ2：企業情報を読み込ませる
プロンプト2を使って、志望企業の情報をAIに分析させます。

### ステップ3：志望動機と想定問答を生成
プロンプト3を使って、3×3志望動機と想定問答を作成します。

### ステップ4：磨き上げる
生成された内容を、**何回も考え直して改善**します（Think Againの考え方）。

### ステップ5：キーワード3つに落とし込む
各質問の回答を**キーワード3つ**に凝縮し、カンペ（PC上部に貼る紙）を準備します。

### ステップ6：実践と振り返り
面接で実際に使ってみて、面接官の反応を観察し、録音を聞き返して改善します。

---

## 注意点

### ✅ 丸暗記はNG
キーワード3つだけ覚えて、自分の言葉でアドリブで話すことで、自然で熱意のある話し方になります。

### ✅ 何回も考え直す
一度作って終わりではなく、何回も見直して改善を重ねることで、質が劇的に向上します。

### ✅ 本命企業は最後に
他社で実戦経験を積んでから、本命企業の面接に臨みましょう。

### ✅ 継続的な改善サイクル
面接を録音し、振り返りを行い、次の面接に活かすサイクルを回しましょう。
