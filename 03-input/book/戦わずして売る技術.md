🔵マーケティングの成果を最大化する「ビジネスモデル」の重要性 → ビジネスモデルが成功の鍵 　→ どんなに優れたWEBマーケティングでも、ビジネスモデルが脆弱だと努力は水の泡になる可能性がある。 　→ まず「儲かるビジネスモデル」を見抜く目を養うべき。 　🔷努力が報われるビジネスにするたった1つの条件 　→ スケーラビリティ（拡大性）の確保 　　・儲かりにくいビジネスモデルの本質は「スケーラビリティ」の低さにある。 　　《例》喫茶店は席数、フリーランスは労働時間やキャパシティが売上の上限を決め、スケーラビリティが低い。 　　《例》Eコマースは在庫さえ確保できれば、売上は青天井に拡大する可能性がある。 　　🅿️ WEBマーケティングのスキルを磨く前に、ビジネスモデルの優劣を見極めることが重要。 　🔷リピート性、仕入れ、オペレーション等のスケーラビリティを阻むもの 　→ リピート性の重要性 　　・リピート性のない商品は、顧客が満足しても翌年の売上がゼロになる可能性がある。 　　《例》「ビリーズブートキャンプ」は初年度195億円の売上も、翌年は9000万円に激減。耐久商品でリピーターがいなかったため。 　→ 定期購入・サブスクリプションモデル 　　・毎月同じ新規顧客獲得数でも、売上が継続的に上がり続ける。 　　・1人の顧客を獲得するコスト（CPO）が高くても、リピート性があれば採算が合う。 　　▶️ 携帯電話の販売がその究極の例で、本体0円でも契約継続による売上があるため採算が合う。⏹ 　　《例》北の達人コーポレーションは、北海道特産品販売（仕入れ・梱包・物流の課題）から健康食品・化粧品の定期購入モデルに転換。結果、同じ苦労で何倍もの利益を生み出せた。 　　🅿️ マーケティング理論やテクニック以上に、ビジネスモデルそのものが売上を大きく左右する。 　🔷「売上の最大化」≠「利益の最大化」 ★ 　→ 売上額と利益額は必ずしも比例しない。 　→ イノベーター理論における顧客層とCPO（顧客獲得コスト）の関係。 　　・イノベーター層：情報感度が高く、CPOが低い。 　　・マジョリティ層：流行や周囲に流されて購入するため、CPOが高くなる。 　　・「収穫逓減の法則」→ 興味がない人に広告でアプローチすると、CPOが上昇し効率が悪化する。 　→ 全体利益の方程式 → 全体利益 ＝ 顧客1人あたり利益（LTV−CPO） × 顧客獲得人数。 　　《例》LTV1万円の場合、CPO4000円で150人獲得した時が最も利益が高く90万円（売上150万円）。CPO6000円で200人獲得した時の方が売上は200万円と大きいが、利益は80万円で低くなる。 　　🅿️ 利益最大化のためには、採算が悪化する手前で投資をやめる必要がある。WEB広告はすぐに停止できるため、この判断が容易。 　　💭 広告費の投入をどこで止めるかという見極めが重要。 　🔷なぜ Amazonはわざわざ自社商品より安い中古品を表示するのか？ 　→ Amazonの戦略的ビジネスモデル 　　・新品が売れた場合と中古品が売れた場合で、Amazonが得る利益はほぼ同じ（約200円）。 　　・中古品の販売は、Amazonに在庫リスク、倉庫コスト、返品リスクがほとんどないという大きなメリットがある。 　　・新品は顧客を引き寄せる「集客ツール」として機能し、中古品や他の出品者による商品が実際の利益源となっている可能性がある。 　　🅿️ 利益は売上規模だけでなく、ビジネスモデル全体、コスト、リスクを総合的に見極めることで、最も効率的な利益を生み出す方法が見えてくる。

　　・課題：25年前、カニ販売（成果報酬5%）は、生命保険（資料請求で1000円）と比較してアフィリエイターにとって魅力が低かった。生命保険はリアルセールスとのハイブリッドで、1顧客あたりの利益がEコマースのカニ販売と根本的に違った。 　　・転換：「カニを売る」から「カニで集める」へ。 　　　① リーチ戦略：バナー広告を「抽選で毎月10名様にカニをプレゼント！」という懸賞企画に変更。 　　　② レスポンス戦略：懸賞応募と同時に無料メールマガジン登録を促す。 　　　③ 結果：メルマガ登録者へ販促メールを毎週配信し、一定数の購入者を獲得。アフィリエイターは「二次収益の自動化」として積極的に宣伝。 　　🅿️ レスポンス戦略が優れていれば、使いにくいリーチ戦略も有効活用できる。 　🔷SNSでバズるリーチ戦略は有効か？ ★ 　→ SNSバズの限界 　　・口コミやSNSでのバズは、売上20億円程度が限界とされている。それ以上には広告や流通戦略が必要。 　→ インフルエンサーマーケティングの落とし穴 　　・タレントやインフルエンサーの起用に過度な期待は危険。 　　・企業案件（PR明記）の投稿は反応が鈍い傾向にある。露骨な宣伝は本能的に嫌われる。 　　・本当に効果が出るのは、心から「良い」と思って発信される場合のみ。 　→ 日本のライブコマースの課題 　　・中国で活況のライブコマースが日本で普及しなかったのは、効果的な「レスポンス戦略」が不在だったため。 　　・日本のインフルエンサーは商品よりも自分を魅力的に見せることを意識し、「人気者の私がお勧めします」というファングッズの位置づけでしか売れていないことが多い。 　　・ライブコマース本来の姿は商品が主役で、演者は脇役。ライブの臨場感が加わって初めて効果的なレスポンス戦略が実現する。 　　・最近では、西野亮廣氏のYouTube番組や、ライブコマースと広告配信を組み合わせたハイブリッド型レスポンス戦略など、日本でも「レスポンス戦略」を語る事業者が現れ始めた。 　　🅿️ 日本でライブコマースが伸び悩んだのは日本人の気質ではなく、効果的な「レスポンス戦略」がなかったことに起因する。

🤔【逆算思考と本質追求が生むヒットの方程式】
🔵マーケティングを突き詰めれば必ず商品開発にたどり着く → マーケティングの極意は「商品開発」にあり 　→ ビジネスには「ハック」と「本質」が存在する 　　・多くの人は「ハック」（うまくいっているやり方を表面的に真似る）に走り、とりあえず一定の成果を得る。 　　・しかし、「ハックにとどまる人」と「本質にたどり着く人」に分かれる。 　　・「ハックにとどまる人」→ WEBマーケティングで成果を出しても商品企画ができない人。 　　・「本質にたどり着く人」→ 商品を売る広告運用を通じて「なぜ売れるのか」「どうすれば売れるのか」を分析し、「こういう条件の商品を作れば売れる」という理解に至る人。 　　《例》バルミューダは「アメトーーク！」で取り上げられた経験をもとに、「どういう商品がテレビで取り上げられるか」を熟知し商品開発をしていると思われる。 　　《例》ライザップは健康食品のEコマース集客ノウハウを活かしてリアル店舗のライザップを開業し、さらに「チョコザップ」を様々な広告手法でテストを重ね、最も拡散力の高い業態として現在の形にたどり着いた。 　　・マーケターは「売れた理由」を「どう伝えたか」というレベルでなく、「商品に何があったから売れたのか」という本質まで昇華する必要がある。 　　・自分が「ハック止まり」か「本質を掴めているか」の判断基準は、担当している業務の1段階上の課題が見えているかどうか。 　　🅿️ マーケティングの本質は商品開発にある。ハックで終わらず、売れた要素を分析し、それを活かしてもっと売れる商品を作るという視点を持つことが重要。
💬圧倒的伝達力を生むキャッチコピーの企画術 → キャッチコピーから逆算して商品企画を設計する 　→ 良いレスポンス戦略には、思わず反応してしまう良いクリエイティブが必要。 　→ 良いクリエイティブは、USPが伝わりやすいものであり、そのためには伝わりやすいUSPを持った商品を作る必要がある。 　　《例》ダウンジャケットの商品企画 　　　・他社よりフィルパワーの高い羽毛を使用しているため「軽くて暖かい」というリーズンUSPがあるが、そのままでは伝わりにくい。 　　　・「500mlペットボトルの半分以下」「卵4つ分」など、具体的な数値や身近なもので軽さを表現するアイデアを検討。 　　　・最終的に「スマホ1台分の軽さのダウンジャケット」というキャッチコピーを発想。 　　　・このキャッチコピーを実現するため、商品の軽量化（150gまで）を検討する（フード取り外し、生地やボタン、ファスナーの材質変更など）。 　　🅿️ 売れる商品企画とは、「まず今、考えられる最適な商品を企画する」→「レスポンス戦略や、クリエイティブを考える」→「そのレスポンス戦略やクリエイティブに合わせて商品企画を改良する」というサイクルである。 　　《例》炭酸泡のオールインワン化粧品 　　　・「泡だから浸透後の乾きが早い」というベネフィットを最大限に伝えるために「10秒スキンケア」というキャッチコピーを考案。 　　　・しかし、実際には浸透に15秒程度かかることが判明。 　　　・そこで「10秒スキンケア」という響くキャッチコピーに合わせて、商品の配合成分を見直し、10秒で浸透するように製品を改良した。 　　🅿️ マーケターの商品企画は「顧客起点」でも「商品成分起点」でもなく、「顧客と商品の接点起点」で考えるべき。
